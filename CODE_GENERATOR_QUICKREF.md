# Code Generator & Parser - Quick Reference

## ?? Quick Access

**Menu**: `Tools ? Code Generator & Parser...`

---

## ?? Generate Code from Structure

1. **Select structure** from dropdown
2. **Check "Include comments"** (optional)
3. **Click "Generate C++ Code"**
4. **Click "Copy to Clipboard"**

? Complete C++ code ready to compile!

---

## ?? Parse Code to Create Structure

1. **Switch to "Parse" tab**
2. **Paste your C++ code**
3. **Click "Parse & Create Structure"**
4. **Structure auto-created and visualized**

? Instant visualization from code!

---

## ?? Supported Code Patterns

### Array
```cpp
std::vector<int> arr = {1, 2, 3, 4, 5};
```

### Linked List
```cpp
Node* head = new Node(10);
current->next = new Node(20);
```

### Binary Tree
```cpp
root = insert(root, 50);
root = insert(root, 30);
```

### Graph
```cpp
g.addNode("A");
g.addEdge("A", "B");
```

## ?? Key Features

? **Bidirectional**: Structure ? Code  
? **Auto-detect**: Recognizes structure type  
? **Compilable**: Generated code runs as-is  
? **Error handling**: Clear parse error messages  
? **Integration**: Works with all UI features

## ?? Tips

1. Include comments when learning
2. Follow pattern examples for parsing
3. One structure type per code paste
4. Rename structures after parsing
5. Test generated code in compiler
